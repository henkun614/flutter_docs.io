### 5.1 文本组件
#### 5.1.1 Text
Text 主要用于显示简单的文本信息,构造方法中包含以下属性:
```dart
this.data, //文本内容
Key key,
this.style,//文本样式
this.strutStyle,
this.textAlign,//文字问题
this.textDirection,//文字方向
this.locale,
this.softWrap,//是否自动换行
this.overflow,//超出最大宽度之后的截取方式
this.textScaleFactor,//缩放因子
this.maxLines,//最大行数
this.semanticsLabel,
this.textWidthBasis,
this.textHeightBehavior,
```
